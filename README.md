# AAA

Godot 4.6 project.

## Working Across Windows and macOS

- Open `project.godot` from the Godot Project Manager.
- Git normalizes text files to LF line endings via `.gitattributes`.
- Generated Godot cache files in `.godot/` are ignored.
- Asset import metadata (`*.import`) and script UID files (`*.uid`) are kept in version control so the project opens consistently on both platforms.
